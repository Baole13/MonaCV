Demo Page with Extracted Navbar

This demo page shows how easy it is to use the extracted navbar component. Just add the navbar placeholder and load the script.

How to use the navbar:

  1. Add <div id="navbar-placeholder"></div> where you want the navbar to appear
  2. Add <script src="load-navbar.js"></script> before the closing body tag
  3. That's it! The navbar will be loaded dynamically from navbar.html
